# Simple HTML Hifi-API player

## What is this?
This is a simple single-file HTML HiFi Music Player. It fetches the original Tidal stream URL using the [hifi-api](https://github.com/uimaxbai/hifi-api) and plays them. I tried to make it as simple as possible, so that everyone understands the code.


## Related Projects:
